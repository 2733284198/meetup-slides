# 第8届Service Mesh Meetup北京站

时间：2019年11月24日（星期六）

地点：北京望京东浦项中心

视频回放：https://tech.antfin.com/community/activities/985

Service Mesh Meetup 是由蚂蚁金服联合 CNCF 官方共同出品，ServiceMesher 社区主办，主题围绕服务网格、Kubernetes 及云原生，在全国各地循环举办的技术沙龙。
